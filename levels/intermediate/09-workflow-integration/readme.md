| | |
|---|---|
| **Difficulty** | ⭐⭐⭐ Intermediate |
| **Time** | 20–25 minutes |

| **Skill** | Chain AI tasks across multiple applications |

---

## Objective

Develop the ability to orchestrate AI-assisted workflows across Microsoft 365 applications, creating efficient multi-step processes for complex deliverables.

---

## Business Context

Most deliverables span multiple applications. This level develops capability to design and execute AI-assisted workflows that connect across Microsoft 365.

## Your Task

Design and execute a workflow that:
1. Gathers information from one or more sources
2. Processes or analyses that information
3. Creates a final deliverable in appropriate format

---

## Example Flow

```
Copilot (gather from email) 
    → Excel (analyse data) 
    → Word (write narrative) 
    → PowerPoint (create presentation)
    → Outlook (distribute)
```

---

## Success Criteria

✅ Workflow produces a cohesive final deliverable  
✅ Each step adds clear value  
✅ Quality maintained throughout the chain  
✅ Total time is less than manual approach
