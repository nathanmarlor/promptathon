## Starter Prompts

### Creating from Scratch

```
Create a presentation about [topic].

Structure:
1. Title slide
2. Executive summary/agenda
3. [Content slides - describe main sections]
4. Recommendations/next steps
5. Q&A slide

Audience: [who]
Purpose: [inform/persuade/decide]
Length: [number] slides maximum
Tone: [professional/inspiring/urgent]
```

### Key Principles

- One message per slide
- Minimal text—headlines + 3 bullets max
- Include speaker notes
- Build toward a clear conclusion

---

## Success Criteria

✅ Each slide makes one clear point  
✅ Flow builds logically to conclusion  
✅ Minimal text—presentation is spoken, not read  
✅ Speaker notes enable confident delivery
