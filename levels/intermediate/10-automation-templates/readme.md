## Your Task

**Step 1:** Identify 2-3 recurring tasks that consume meaningful time

**Step 2:** Design prompt templates for each task

**Step 3:** Test and refine until outputs are consistently usable

**Step 4:** Document for reuse

---

## Template Structure

```
TEMPLATE NAME: [Descriptive name]

TRIGGER: [When to use this]

REQUIRED INPUTS:
- [INPUT_1]: [description]
- [INPUT_2]: [description]

PROMPT:
Context: [Fixed framing + {placeholders}]
Language: [Standard tone/style]
Examples: [Standard structure]
Audience: [Who this is for]
Request: [Standard deliverable spec]

QUALITY CHECKLIST:
- [ ] [Check 1]
- [ ] [Check 2]
```
