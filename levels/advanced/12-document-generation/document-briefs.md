# Document Generation Briefs

Specifications for practicing document generation with Claude.

---

## Brief 1: Board Paper

**Document Type:** Board paper for governance meeting  
**Format:** Word document  
**Length:** 3-4 pages plus appendices

**Required Sections:**
1. Paper reference and title
2. Executive summary (half page)
3. Purpose and recommendation
4. Background and context
5. Analysis and options
6. Recommendation with rationale
7. Financial impact
8. Risk considerations
9. Next steps and timeline
10. Appendices

**Sample Prompt:**
```
Create a board paper template for [topic: e.g., "New Market Entry Proposal"].

Purpose: Seeking board approval for [decision].
Recommendation: [Approve/Reject/Defer + brief rationale]

Include:
- Professional formatting suitable for board distribution
- Executive summary that stands alone
- Clear recommendation section
- Risk assessment table
- Financial summary table
- Placeholder text showing what goes in each section

Format as downloadable Word document.
```

## Brief 2: Business Case

**Document Type:** Investment proposal  
**Format:** Word document  
**Length:** 5-7 pages

**Required Sections:**
1. Executive summary
2. Problem/opportunity statement
3. Proposed solution
4. Options considered
5. Cost-benefit analysis
6. Implementation approach
7. Timeline and milestones
8. Resource requirements
9. Risks and mitigations
10. Recommendation

**Sample Prompt:**
```
Create a business case document for [initiative name].

Investment: £[amount]
Expected return: [description]
Timeline: [duration]

Include:
- Executive summary (1 page)
- Financial analysis with NPV/ROI table
- Implementation timeline (visual if possible)
- Risk register table
- Clear recommendation section

Format as Word document ready for finance review.
```

## Brief 3: Strategy Presentation

**Document Type:** PowerPoint presentation  
**Format:** .pptx file  
**Length:** 12-15 slides

**Required Slides:**
1. Title slide
2. Executive summary
3. Context/Why now
4. Strategic objectives (3-4)
5-8. One slide per objective with initiatives
9. Roadmap/timeline
10. Investment requirements
11. Expected outcomes
12. Risks and dependencies
13. Next steps
14. Q&A

**Sample Prompt:**
```
Create a strategy presentation for [strategy name].

Key messages:
1. [Strategic objective 1]
2. [Strategic objective 2]
3. [Strategic objective 3]

Audience: [Executive team/Board/All employees]
Duration: [X] minutes

Requirements:
- One clear message per slide
- Minimal text (headlines + 3 bullets max)
- Speaker notes for each slide
- Professional design suitable for senior audience

Export as PowerPoint file.
```

## Brief 4: Excel Model/Template

**Document Type:** Financial model or tracking template  
**Format:** .xlsx file

**Required Components:**
- Instructions tab
- Inputs tab (with validation)
- Calculations tab
- Dashboard/summary tab
- Sample data showing usage

**Sample Prompt:**
```
Create an Excel template for [purpose: e.g., "Project Budget Tracking"].

Requirements:
Tab 1 - Instructions: How to use this template
Tab 2 - Inputs: Budget line items, actual spend, dates (with dropdown validation)
Tab 3 - Calculations: Variance calculations, YTD totals, forecasts
Tab 4 - Dashboard: Summary KPIs, traffic light indicators, key charts

Include:
- Data validation on input fields
- Conditional formatting for variances
- Sample data demonstrating usage
- Protected formulas where appropriate

Export as Excel file.
```

## Brief 5: Project Charter

**Document Type:** Project initiation document  
**Format:** Word document  
**Length:** 2-3 pages

**Required Sections:**
1. Project name and sponsor
2. Project overview (1 paragraph)
3. Objectives and success criteria
4. Scope (in scope / out of scope)
5. Key deliverables
6. Timeline and milestones
7. Budget summary
8. Team and governance
9. Key risks
10. Approval signatures

**Sample Prompt:**
```
Create a project charter for [project name].

Sponsor: [name]
Objective: [what the project will achieve]
Timeline: [start] to [end]
Budget: £[amount]

Include:
- Clear scope boundaries (in/out)
- SMART success criteria
- Milestone table
- RACI matrix placeholder
- Risk summary table
- Signature block for approvals

Format as Word document ready for project kickoff.
```
