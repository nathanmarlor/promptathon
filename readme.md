| | |
|---|---|

| **Format** | Hands-on exercises with increasing complexity |
| **Tools** | Microsoft 365 Copilot, Claude |

## Programme Structure

| Tier | Levels | Focus | Tools | Time |
|:-----|:------:|-------|:-----:|:----:|

## The CLEAR Framework

A structured approach to writing effective prompts.

| Letter | Component | Ask Yourself |
|:------:|-----------|--------------|
| **C** | Context | What's the situation? |
| **L** | Language | What tone and style? |
| **E** | Examples | What should it look like? |
| **A** | Audience | Who is this for? |
| **R** | Request | What exactly do I want? |

### Template

```
Context: [situation/background]
Language: [tone/style]
Examples: [format to follow]
Audience: [who, what they know]
Request: [deliverable, length]
```

## Session Facilitation

### Recommended Flow

**45-minute session:** Complete Levels 1–5 (Foundation)

**Half-day session:** Complete Levels 1–10 (Foundation + Intermediate)

**Full workshop:** Complete all 15 levels across multiple sessions

### Tips for Facilitators

- Encourage participants to use their own work materials where possible
- Sample data is provided in each level folder for those who need it
- Allow time for discussion between levels
- Capture insights and effective prompts for team reference
